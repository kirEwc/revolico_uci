# POTENTIAL ERRORS — Registro de errores complejos

> Cuando encuentres un error que no puedas resolver rápidamente o que requiera una solución ingeniosa, regístralo aquí con el formato siguiente. Esto ayuda a no repetir la misma investigación.

---

## Formato de registro

```
## [YYYY-MM-DD] Título corto del error

**Contexto**: ¿Qué estabas haciendo?
**Error**: Mensaje o stack trace relevante
**Causa raíz**: Lo que realmente lo provocaba
**Solución**: Cómo lo resolviste
**Archivos afectados**: ruta/al/archivo.tsx
```
